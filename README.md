Geothermal Reservoir Asset Manager 🌍
Geothermal Reservoir Evaluation Automation Project. This pipeline transforms raw drilling data into professional, decision-ready engineering reports.
​Key Features:
​Data Pipeline: Ingestion from MySQL databases and version control via GitHub.
​Data Science: Automated signal conditioning (despiking, linear interpolation).
​Physics: Heat-In-Place (HIP) estimation based on USGS stochastic formalisms.
​Interface: Interactive Streamlit dashboard for real-time well log analysis.
​Reporting: Automated generation of professional PDF technical reports using ReportLab.
